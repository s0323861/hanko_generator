## Hankoツクール

Web認印を作成するツールです。

デザインはBootstrap3、Font Awesome Iconsを使っています。

フリーフォント(ダサ字フォント、にくまるフォント)も使わせていただきました。


### Usage

  - upload all files to the web server


### Features

  - written in PHP
